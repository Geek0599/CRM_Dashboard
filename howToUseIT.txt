КОМАНДИ ЗАПУСКУ:
- npm run dev => Запуск збірки в режимі розробки;
- npm run fonts => Запуск конвертації шрифтів та запис їх до файлу src/fonts/fonts.scss 
- npm run sprite => Запуск збірки svg спрайту 
- npm run build => Запуск збірки для фінальної збірки проекту для видачі клієнту
- npm run zip => Запуск збікри для створення zip архіву проєкту
- npm run deploy => Запуск збірки для відправки скомпільованих файлів на віддалений ftp сервер


ПІДКЛЮЧЕННЯ ШРИФТУ ЗАВАНТАЖЕНОГО З ІНТЕРНЕТУ:
1) Завантажити потрібний шрифт в папку src/fonts в форматі otf або ttf 
2) Перейменувати їх до стандарту, щоб збірка могла автоматично їх підключити та розпізнати за насиченістю (наприклад light(300), bold(700), medium(500) )
3) Стандарт найменування шрифта - Приклади Або || Або =>   OpenSand-bold.otf || OpenSand-700.otf
                                   								  SFProDisplay-Light.otf || SFProDisplay-300.otf
                                   								  Bebas-Regular.otf || Bebas-400.otf
                                   								  gilroy-semibold.otf || Gilroy-Semibold.otf || Gilroy-600.otf

	Назва має бути без пробілів => Перше слово - це Назва Шрифта, далі тире, і далі Насиченість шрифту - можна обо словами або цифрами;
4) Запустити збірку командою - npm run dev , або конкретною командою npm run fonts;

ПІДКЛЮЧЕННЯ ІКОНОЧНОГО ШРИФТУ:
1) Вивантажити всі необхідні іконки (тільки формат SVG);
2) Завантажити іконки в новий сет на сайті icomoon.io ;
3) Прибрати в іконках колір, перейменувати їх (за неохідністю - для зручності використання);
4) Налаштування - Fonts Name => icons, Class Prefix => _icon-, Class Suffix => , Support IE8;
5) Завантажити zip архів з сайту;
6) Взяти з архіву файлик - icons.ttf та помістити його до папки src/fonts
7) Взяти з файлика style.css - класи з before (де прописаний код іконки) та вставити їх до файлику scss/fonts/icons.scss в кінець;
8) Розкоментувти в файлику style.scss рядки імпорту "fonts/fonts" , "fonts/icons"
9) Запустити збірку командою - npm run dev;

ПІДКЛЮЧЕННЯ SVG SPRITE
1) Створити папку svgicons в папці src та завантажити туди всі svg іконки;
2) Запустити команту npm run sprite
3) Збірка створить файл img/icons/sprite.svg;
4) Приклад використання => <svg class="sprite-icon">
										<use xlink:href="@img/icons/sprite.svg#calendar"></use>
									</svg>

СТВОРЕННЯ zip АРХІВУ
1) Запуск команди ==> npm run zip
2) Створиться Zip архів в корінь папки нашого проєкту 

ВІДПРАВКА ФАЙЛІВ НА СЕРВЕР ЗА ДОПОМОГОЮ FTP
1) Заповнити конфігураційний файл config/gulp-settings.js // Налаштування FTP підключення
2) Заповнити в тому же файлі ==> ftp: `` // Шлях до потрібної папки на віддаленому сервері.
3) Запустити команду ==> npm run deploy
